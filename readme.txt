== Description ==

LoudReply allows you to collect customer feedback instantly from your website for free. 

== Installation ==
1. Sign up for a free LoudReply account at https://LoudReply.com. This will allow you to grab a secret user ID. 

2. Ensure you have WooCommerce installed and set-up for the site you wish to collect feedback. 

3. Upload the folder 'woocommerce-LoudReply' to the '/wp-content/plugins/' directory or install it from Within Wordpress

4. Activate 'LoudReply Surveys' through the 'Plugins' menu in WordPress

5. Update the plugin settings through the 'WooCommerce' menu in WordPress

That's it! Enjoy using LoudReply for WooCommerce! If you have any questions feel free to write to us at support@loudreply.com.
